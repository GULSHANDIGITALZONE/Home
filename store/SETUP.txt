WHAT YOU NEED TO MAKE IT REAL

1. Admin username
   Put in backend `.env`
   `ADMIN_USERNAME=your_admin_name`

2. Admin password
   Put in backend `.env`
   `ADMIN_PASSWORD=your_strong_password`

3. Random secret
   Put in backend `.env`
   `JWT_SECRET=any_long_random_secret`

4. Your frontend URL
   Put in backend `.env`
   `CLIENT_ORIGIN=https://your-frontend-url.com`

5. Razorpay Key ID
   Put in backend `.env`
   `RAZORPAY_KEY_ID=rzp_live_xxxxx`

6. Razorpay Key Secret
   Put in backend `.env`
   `RAZORPAY_KEY_SECRET=your_secret`

7. Enable Razorpay
   Put in backend `.env`
   `RAZORPAY_ENABLED=true`

8. Update frontend API URL
   In `store-config.js`
   `apiBaseUrl: "https://your-backend-url.com/api"`

9. Update frontend Razorpay key
   In `store-config.js`
   `payment.razorpayKey`

10. Enable secure verify mode
   In `store-config.js`
   `payment.enableServerVerification = true`

10b. Optional external integrations
   In backend `.env`
   - `BREVO_API_KEY`
   - `BREVO_SENDER_EMAIL`
   - `RESEND_API_KEY`
   - `RESEND_SENDER_EMAIL`
   - `WHATSAPP_TOKEN`
   - `WHATSAPP_PHONE_NUMBER_ID`
   - `CLOUD_DOWNLOAD_BASE_URL`
   - `CLOUD_DOWNLOAD_TOKEN`

11. Replace your support email and WhatsApp
   In `store-config.js`
   `support.orderEmail`
   `support.orderWhatsapp`

11b. Optional backup payments
   In `store-config.js`
   - `backupPayment.upiId`
   - `backupPayment.cashfreeLink`
   - `backupPayment.phonepeLink`

12. Replace product info
   In `store-config.js` or backend `data/products.json`
   Change:
   - id
   - title
   - details
   - amount
   - image
   - type
   - actionLabel
   - actionUrl
   - features

13. Firebase project create karo
   Firebase Console me project banao.

14. Authentication enable karo
   Firebase Console → Authentication → Sign-in method
   Enable:
   - Google
   - Phone

15. Authorized domains add karo
   Add your frontend domain:
   - gdz.services
   - your preview/local domain if needed

16. Firebase config values copy karo
   In `store-config.js`
   Replace:
   - firebase.apiKey
   - firebase.authDomain
   - firebase.projectId
   - firebase.storageBucket
   - firebase.messagingSenderId
   - firebase.appId

HOW TO RUN BACKEND

1. Open terminal in `store/backend`
2. Run:
   `npm install`
3. Copy `.env.example` to `.env`
4. Fill real values
5. Run:
   `npm start`

IMPORTANT

- Frontend-only payment secure nahi hota.
- Real secure unlock ke liye backend verification already scaffold me diya gaya hai.
- Admin panel `admin.html` se products aur orders control kar sakte ho.
- Aapne jo `razorpay.me/@...` diya hai wo payment link hai, Key ID nahi.
- Abhi project me payment link fallback set hai.
- Fully secure live Razorpay checkout ke liye actual `rzp_live_...` Key ID abhi bhi chahiye.
